How to use?

Enter your task id, description and duration.(eg 2h for 2 hours, 10m for minutes, 30s for 30 seconds)

Press enter or submit button

Click each task individually to start task timer

Double click task to complete them

The task for which duration is over, appear in expired column and can be reentered by clicking on them
